# 失明症（Blindness）

**失明症**是一款 Minecraft Fabric 体验型模组。玩家无法直接看见世界，需要通过导盲杖、声音、
声纹、环境轮廓和危险提示判断周围情况。

世界保持纯黑，不通过降低亮度来模拟。玩家依靠触觉式探测和声音感知获取环境信息。

> **重要说明**：本模组以游戏化方式表现部分重度视力障碍者可能使用的环境感知方式，不代表所有盲人或视障人士的真实体验，也不应被视为医学或生活体验的完整模拟。

## 核心功能

- **完全黑暗的世界**：启用失明症后世界全黑，背包、GUI 和菜单正常显示，暂停画面后方保持纯黑。
- **导盲杖接触探测**：使用导盲杖敲击或扫动方块，短暂显现被击中方块和相邻方块的真实轮廓。
- **真实模型高级轮廓**：短草、花、火把、栅栏、楼梯、半砖等显示与方块模型一致的轮廓，而非简单方框。
- **悬崖和危险落差警告**：导盲杖探测前方时检测 2 格以上落差、岩浆或虚空，给出声音和文字警告。
- **摔倒系统**：高速移动撞到障碍物、生物或未探测的落差时可能摔倒，摔倒后短暂失去操作能力并需要起身。
- **生物声音声纹**：当生物实际发出声音时，屏幕显示对应方向和高度的声纹标记，声纹固定在发声瞬间的位置。
- **敌对生物模糊警告**：检测到附近敌对生物时给出不暴露数量、名称或距离的危险提示。
- **可配置监听范围**：生物声音监听范围（0/1/2 区块半径）和声纹触发方块轮廓范围均可调整。
- **信息类模组兼容检查**：进入世界前检测地图、小地图和方块信息 HUD 模组，保证体验完整性。
- **Mod Menu 与游戏内配置**：通过 Mod Menu 或快捷键 `B` 打开 owo-lib 配置界面，调整视觉、声音和辅助选项。

## 导盲杖

### 配方

|   |   |   |
|---|---|---|
| 铁锭 | 白色羊毛 |   |
|   | 木棍 |   |
|   | 木棍 |   |

合成 1 个导盲杖。

### 使用方式

- **短按（右击后立即松开）**：敲击前方 4 格内的方块，被命中方块及直接正交相邻方块短暂显示真实模型轮廓。
- **长按（按住右键）**：在一秒内沿水平方向连续摆动接触 4 次（左右各两次），可探测更大范围。

### 轮廓特性

- 轮廓使用方块真实的渲染模型，短草、花、栅栏、楼梯等显示实际形状。
- 中心方块轮廓更亮更粗，相邻方块稍暗稍细，全部带光晕效果。
- 轮廓保持约 5 秒后淡出，淡出前新接触会刷新计时。
- 轮廓尊重遮挡，不会显示墙后的方块或矿物。
- 未命中方块时不显示任何轮廓。

## 声纹系统

- **只有声音才产生声纹**：生物实际发出声音时屏幕出现声纹标记，静止未发声的生物不会持续显示。
- **声纹位置固定**：声纹记录发声瞬间的位置和高度，生物移动后旧声纹留在原位淡出。
- **不追踪实体**：不显示生物名称、血量、装备、数量、类型和精确距离。
- **屏幕边缘声纹**：位于视野外的声音会在屏幕边缘显示半圆标记。
- **墙后模糊**：被墙壁遮挡的声音声纹更模糊，碎片化程度更高，但不显示墙后方块。
- **身体锚定**：声纹标记在生物身体高度（非脚下方块），脚步约 35% 高度，叫声约 65% 高度。
- **默认监听范围**：3×3 区块（玩家所在区块和相邻 8 个区块）。
- **可配置范围**：监听区块半径可设为 0（仅当前区块）、1（3×3）、2（5×5），声音触发的环境轮廓范围可设为 0～4。

## 悬崖警告

- 需使用导盲杖进行探测，空手站立时不会获得地形雷达。
- 沿水平朝向在 4 格内采样，检测实际方块碰撞形状的可行走顶面。
- 1 格落差通常安全；2～3 格给出双重提示音和"前方有明显落差"文字；4 格及以上、岩浆、火焰或虚空给出三连警告。
- 不显示悬崖精确深度、坐标或安全路径，不移动或转向玩家。
- 同一边缘和朝向有 2 秒冷却防止重复警告。

## 摔倒机制

- 疾跑时撞到障碍物、发生水平碰撞或遇到未探测的突然高度变化可能摔倒。
- 撞到不同体型生物时风险不同（大型生物风险高于小型生物）。
- 慢走、蹲行和已用导盲杖探测过的路径可大幅降低绊倒风险。
- 摔倒后约 5 秒失去移动和操作能力，随后自动起身。
- 摔倒在草地、雪地或羊毛上不会受伤；在硬地面（石头、金属）上可能受到少量伤害。
- 平地正常行走不会无理由随机摔倒，摔倒后有冷却保护。

## 安装方式

1. 安装 Minecraft 1.21.1。
2. 安装兼容版本的 [Fabric Loader](https://fabricmc.net/)（≥ 0.19.3）。
3. 确保使用 Java 21。
4. 安装所有必需前置模组（见下方依赖表）。
5. 将本模组和所有前置模组放入 `mods` 文件夹。
6. 启动游戏。

## 操作方式

| 操作 | 按键 |
|------|------|
| 使用导盲杖 | 右键 |

按键可在 Minecraft 控制设置 → 失明症分类中重新绑定。

## 指令

| 指令 | 说明 |
|------|------|
| `/blindness enable` | 开启失明症体验 |
| `/blindness disable` | 关闭失明症，恢复正常视觉和字幕 |
| `/blindness status` | 查看当前状态（开启/关闭、视觉模式、导盲杖熟练度、累计摔倒次数） |
| `/blindness reset` | 重置持久和临时数据（需要 OP 2 级或创造模式） |

单人世界和多人游戏均可使用。`disable` 用于无障碍退出和排错。

## 配置说明

通过 **Mod Menu → 失明症 → 配置** 或按 `B` 键打开。

主要配置项：

- **视觉**：轮廓粗细、亮度、光晕、淡入淡出时间；菜单后方纯黑开关；手持导盲杖保持可见。
- **辅助**：镜头晃动强度；关闭第一人称摔倒倾斜；显示教程。
- **声音感知**：生物声纹开关；监听区块范围（0/1/2）；声音轮廓范围（0～4）；屏幕外声纹；声纹尺寸/亮度/时长；墙后模糊强度；最大同时显示数量。
- **兼容性**（修改后需重启）：地图/小地图拦截；方块信息 HUD 拦截；世界透视拦截；自定义追加/忽略 Mod ID。

其中监听区块范围和声音轮廓范围会同步到服务端并持久化到玩家数据。

## 不兼容模组

- **DashLoader**：当前版本不支持。其着色器缓存机制会干扰 Veil 的着色器与后处理管线初始化，可能导致失明黑屏失效、探测轮廓异常或客户端报错。

### 受限制的模组

进入世界前会检测以下类型的模组。

### 地图 / 小地图

`xaerominimap`、`xaeroworldmap`、`journeymap`、`voxelmap`、`ftbchunks`、`antiqueatlas`、`antique_atlas`、`antique_atlas_4`、`map_atlases`、`mapatlases`

### 方块信息 HUD

`jade`、`wthit`、`hwyla`、`waila`、`theoneprobe`

### 世界透视（可配置）

可在配置中追加自定义 Mod ID。

> 此系统是体验完整性检查，不是安全级反作弊。客户端报告在理论上可以被修改或伪造。

## 允许使用的查询模组

以下类型不会被限制：JEI、REI、EMI 及其他仅用于配方和物品用途查询的模组。硬允许 ID 为 `jei`、`roughlyenoughitems`、`roughlyenoughitems-api`、`emi`。
